# IES Comuneros de Castilla

Web de admisión y matrícula 2026-2027 con chatbot integrado (n8n).

## Tecnología
- HTML/CSS/JS estático (un único archivo `index.html`)
- Chatbot: [@n8n/chat](https://www.npmjs.com/package/@n8n/chat) conectado a webhook propio

## Despliegue
Desplegado en [Vercel](https://vercel.com). Cualquier push a `main` actualiza la web automáticamente.
